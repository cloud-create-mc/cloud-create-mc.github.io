Як встановити

Обери свій тип гри, щоб переглянути відповідну інструкцію.

1

Встанови лаунчер

Завантаж та встанови Modrinth App або Prism Launcher. Вони чудово підтримують сучасний формат модпаків .mrpack. Також підійде офіційний лаунчер CurseForge.

2

Скачай модпак

Натисни золотисту кнопку завантаження вище, щоб завантажити файл збірки .mrpack на свій комп'ютер.

3

Імпортуй збірку

У вашому лаунчері натисни кнопку Create Profile (або Add Instance) → вибери вкладку From File (З файлу) → вкажи завантажений файл .mrpack та зачекай закінчення імпорту.

Порада (для Modrinth App): Якщо у вас вже встановлено цей лаунчер, ви можете просто двічі клікнути по завантаженому файлу .mrpack у вашому файловому менеджері — збірка автоматично імпортується та завантажиться.

4

Виділи пам'ять та запусти

Бажано виділити 4–6 ГБ оперативної пам'яті для стабільної роботи у налаштуваннях Java вашого лаунчера. Після цього запускай гру, переходь у Multiplayer та підключайся за IP: .

1

Встанови лаунчер

Скачай та встанови лаунчер, який підтримує автономний режим (офлайн акаунти), наприклад Legacy Launcher або TLauncher.

2

Скачай архів модпаку

Натисни синю кнопку завантаження вище, щоб завантажити .zip-архів зі збіркою на комп'ютер.

3

Встановлення модпаку

Автоматично (Legacy Launcher): Перейди у створення профілю → кнопка Імпортувати з архіву → вибери завантажений .zip.

Вручну (TLauncher / інші): Встанови версію NeoForge 1.21.1, відкрий папку збірки у лаунчері та розпакуй архів. Тобі потрібно замінити або додати до папки збірки папки з .zip-архіву: mods, resourcepacks та kubejs.

4

Виділи RAM та грай

Перейди в налаштування лаунчера та виділи 4–6 ГБ оперативної пам'яті для стабільної роботи. Запускай гру, заходь у гру по мережі та додавай сервер за IP: .
